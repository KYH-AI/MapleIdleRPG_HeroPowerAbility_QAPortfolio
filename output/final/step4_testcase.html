카테고리별 테스트 현황

CATEGORY 01
UI 출력 — 용사의 힘 / 어빌리티
  • 용사의 힘 메인 창 출력 상태 확인
  • 어빌리티 패널 창 출력 상태 확인
  • 재화(명예의 훈장, 다이아) 아이콘 및 수치 출력 확인
  • 슬롯 구성 요소 (옵션명, 수치, 등급) 출력 정합성 검증
  • 용사의 힘 단계 및 슬롯 개방 상태 출력 확인
Pass 100% v1.9.2 2026/05/12
CATEGORY 02
UI 상호작용 — 용사의 힘 / 어빌리티
  • 용사의 힘 가이드 팁 버튼 상호작용 확인
  • 재화 버튼 탭 및 팝업 출력 상호작용 확인
  • 어빌리티 승급 버튼 및 조건 미충족 토스트 메시지 확인
  • 패널 스크롤 (위/아래) 상호작용 확인
  • 슬롯 잠금/해제 버튼 상호작용 및 상태 반영 확인
Pass 100% v1.9.2 2026/05/13
CATEGORY 03
어빌리티 옵션 출력 확인
  • 어빌리티 변환 레벨(1~20)별 출현 가능 옵션 출력 검증
  • 각 옵션 최솟값/최댓값 출력 정합성 확인
  • 등급(노말~미스틱)별 옵션 풀 변화 시점 확인
  • 레벨 5/8 분기점에서의 신규 옵션 해금 정합성 검증
Pass 100% v1.9.2 2026/05/14
